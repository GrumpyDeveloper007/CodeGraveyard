3D Object Texture Mapper


Keys -

T - Texture On/Off
A - Alpha On/Off

PageUP - Change Current Point +
PageDOWN - Change Current Point -

Insert - Change Texture +
Delete - Change Texture -

Cursor Keys - Rotate object
Shift Cursor Keys - Rotate object fast

Esc - Exit

NOTE: This program requires DX5 and will not work on WinNT machines.
