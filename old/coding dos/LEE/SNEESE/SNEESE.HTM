Ok, this page is to serve as a download point for SNEeSe the SNES emulator with extra e's. I hope to have a better looking design for this in a few days, for now this page is here to prove I am serious about this. Oh, and no there are no downloads of ROMS here and there never will be, dont ask, dont email just go elsewhere!

DJGPP V2.0 is being used as the main compiler cos it is very nice, although I am not happy about the AT&T format assembler. At present the whole project core is pure C++, so it aint gonna be fast just easily maintainable.

SNEeSe Version ?????

Well Not Yet. Maybe soon, for now here is a list of features that currently exist, erm all 3 of them ;-) If you know about this stuff you probably dont need an emulator but it looks impresive to me!

  • DMA operations 0 & 1. Although at present only channel 0 is implemented.
  • 40 of the 256 opcodes of the 65c816 are implemented (Only 1 ROM works 100%).
  • NMI interrupt works although the timing may not be correct ?
  • SNES Screen MODE 1 is supported, 3 is almost there and 0 has not been looked at. Dont even ask about Mode 7.
  • BKG Scroll Registers 0-3 are implemented (I havent done Mode 0 yet so BKG4 is not in use).
  • At present I have no APU skippers, I am unsure how to go about doing this, so the 1 ROM that works is a demo ROM (Yoshi's to be precise), and I have simply NOPed his JSR music.
  • Joypad 1 Emulation - currently the keys are fixed but that can be changed easily enough.
